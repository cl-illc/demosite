# Add a member

1. Create a new md file under `_members/`, start by copying from any existing member;
2. Edit the following 
    * title: member's name (the field is called `title` because that's what the `page` layout expects, sorry about that)
    * description: use, for example, to share contact information (eg, office number)
    * img: link to an image under `assets/img/members/{last name}.jpg` (png is also fine; use the complete path);
    * redirect: use this in case you want the tile to redirect to a personal website, leave it blank otherwise (in which case you can use the rest of the md file, after `---` that is, to share any information you like);
    * category: one of `TODO`
3. Then push the changes (or open a PR):

```bash
git commit -a -m 'new member - name'
git push
```
